# 📊 Litigation 3P Limits Audit Dashboard

## Overview

The **Litigation 3P Limits Audit Dashboard** provides visibility into Third-Party (3P) limits audit activities and pending records. It enables stakeholders to monitor audit status, identify incomplete records, and analyze trends across multiple years and months.

The dashboard is designed to support operational monitoring and ensure timely completion of documentation and information requirements.

---

## Features

- 📅 Year and Month filtering
- 📈 Monthly trend analysis
- 📊 Year-over-year audit comparison
- 🥧 Pending audit distribution analysis
- 📋 Detailed transactional table
- 🔎 Interactive filtering and drill-down capabilities
- 📌 KPI cards for quick status monitoring

---

## Key Metrics

| Metric | Description |
|----------|-------------|
| Blank 3P-1 Field | Records with missing 3P-1 values |
| N/A | Records categorized as Not Applicable |
| Pending Dec Page Confirmation | Records awaiting declaration page confirmation |
| Pending Information | Records requiring additional information |
| Search Requested / Need to Request | Records requiring document searches |

---

## Dashboard Components

### KPI Summary Cards

Provides high-level counts for:

- Blank 3P-1 Field
- N/A
- Pending Dec Page Confirmation
- Pending Information
- Search Requested / Need to Request

---

### Litigation 3P Trend

Displays monthly trends of audit categories, allowing users to:

- Identify spikes and patterns
- Monitor monthly activity
- Compare category movements over time

---

### Yearly 3P Limits Audit Comparison

Compares audit counts by year to analyze:

- Overall audit volume
- Category distribution
- Year-over-year changes

---

### Pending Audit Comparison

Pie chart showing the distribution between:

- Pending Declaration Page Confirmation
- Pending Information

This helps prioritize outstanding items.

---

### Detailed Audit Table

Provides record-level visibility by date, including:

- Blank 3P-1 Field
- N/A
- Pending Dec Page Confirmation
- Pending Information
- Search Requested / Need to Request

---

## Filters

The dashboard supports:

- **Year Selection**
- **Month Selection**

Allowing users to perform targeted analysis and compare audit activity across periods.
